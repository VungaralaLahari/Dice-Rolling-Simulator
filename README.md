# Dice Rolling Simulator 

This is a console-based Dice Rolling Simulator built using Python.  
The program allows the user to roll a dice and get a random number between 1 and 6.

The project demonstrates basic function usage, loops, and user input handling.

---

## Features
- Roll a dice
- Generate random dice values
- Menu-driven program
- Option to exit the program

---

## Concepts Used
- Functions
- While loop and conditional statements
- Random module
- User input handling

---

## How to Run
1. Open the Python file.
2. Run the program using Python.
3. Follow the on-screen menu instructions.

```bash
python Dice_Rolling_Simulator.py
